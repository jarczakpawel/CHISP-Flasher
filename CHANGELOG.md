# Changelog

## 1.1.0

- Added CLI with text and JSON output
- Added Python API for integration with external tools and IDEs
- Added JSONL event stream support for CLI integrations
- Added resolve, doctor, project, list, detect, smart-detect, read-config, write-config, erase, verify and flash CLI commands
- Improved native USB detection, selector refresh handling and expected identify logging
- Improved identify candidate handling for CH32X035 and CH32V30x-family targets
- Improved GUI USB device field sync
- Increased UART programming timeouts for better flash stability
